# HealthyHub

HealthyHub is a web application for tracking nutrition and hydration. It calculates daily calorie and water intake based on user goals and allows logging food and water consumption.
This project demonstrates practical skills in manual testing, test design, and bug reporting.

## What I did:
- Performed manual testing of the web application
- Created a test checklist
- Designed and executed test cases in TestRail
- Reported bugs in Jira

## Tools used:
- Jira
- TestRail
